Class: Mapping FfAA.py

An enhanced dict. Has some properties of list.

This class is started from a little example given somewhere in the Python docs (guess it's where the built-in objects are declared to be subclassable).

It is now something between a list and a dict.

>>> a = mydict() >>> a['test'] = thingy >>> a['zzz'] = aap >>> print a[0] thingy >>> a['est'] aap

WARNING

Though this would be extremely welcome, Mapping is not a subclass of Base.
