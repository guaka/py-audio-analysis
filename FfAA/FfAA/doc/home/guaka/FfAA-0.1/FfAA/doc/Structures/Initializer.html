add_files (
        self,
        files_string,
        accepted_extensions=None,
        )

Initialize sound structures from directory or from glob.

For example:

add_files("/data/music//")

Only the files that have accepted_extensions are taken from the files that fulfill the pattern.

Doesn't handle ~ directories yet.
